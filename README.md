# VERDICT

VERDICT is a standalone decision framework built to tackle buying decision paralysis.

It compares options using AHP pairwise weighting, TOPSIS ranking, minimax regret, and sensitivity analysis. It runs entirely in the browser and does not use AI or external APIs.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The GitHub Pages deployment workflow builds the app and publishes it from the generated `dist` output.
